Basic
• Alt+Enter – Spawn terminal (the default terminal is st; run man st for more.)
• Alt+Shift+Enter – Show/hide dropdown terminal
• Alt+BackSpace – Shutdown menu
• Alt+* – Show/hide dropdown calculator
• Alt+q – Close window
• Alt+d – dmenu (For running commands or programs without shortcuts)
• Alt+D – passmenu (password manager)
• Alt+\ – Select an emoji to copy to clipboard
Tags/Workspaces
• Alt+(Number) – Go to that number tag
• Alt+Shift+(Number) – Send window to that tag
• Alt+Tab – Go to previous tag (may also use \ for Tab)
• Alt+g – Go to left tag (hold shift to send window there)
• Alt+; – Go to right tag (hold shift to send window there)
• Alt+Left/Right – Go to another display
• Alt+Shift+Left/Right – Move window to another display
Window navigation
• Alt+Space – Make selected window the master (or switch master with 2nd)
• Alt+o/O – Increase/decrease the number of master windows
• Alt+j/k – Cycle thru windows by their stack order
• Alt+h/l – Change width of master window
• Alt+z/x – Increase/decrease gaps (may also hold Alt and scroll mouse)
• Alt+a – Toggle gaps
• Alt+A – Gaps return to default values (may also hold Alt and middle click)
• Alt+b – Toggle statusbar (may also middle click on desktop)
• Alt+f – Fullscreen mode
• Alt+F – Floating (AKA normie) mode
• Alt+s – Make/unmake a window “sticky” (follows you from tag to tag)
Window layouts
• Alt+t – Tiling mode (active by default)
• Alt+T – Bottom stack mode (just like tiling, bus master is on top)
• Alt+y – Fibbonacci spiral mode
• Alt+Y – Dwindle mode (similar to Fibbonacci)
• Alt+u – Master on left, other windows in monocle mode
• Alt+U – Monocle mode (all windows full screen and cycle through)
• Alt+i – Center the master window
• Alt+I – Center and float the master window
• Alt+Shift+Space – Make a window float (move and resize with Alt+left/right click).
Programs
• Alt+w – Search the web
• Alt+W – nmtui (for connecting to wireless networks)
• Alt+e – neomutt (email)
• Alt+E – abook (addressbook)
• Alt+r – lf (file explorer)
• Alt+R – htop (task manager, system monitor that R*dditors use to look cool)
• Alt+n – VimWiki (note taking tool)
• Alt+N – newsboat (RSS reader)
• Alt+m – ncmpcpp (music player)
• Alt+c – calcurse (calendar)
Media
• Alt+p – Pause music
• Alt+M – Mute sound
• Alt+- – Volume down
Function keys
• Alt+F1 – Show this document
• Alt+F2 – Watch tutorial videos on a subject
• Alt+F3 – Select screen/display to use
• Alt+F4 – pulsemixer (audio mixer)
• Alt+F5 – Upgrade system
• Alt+F6 – Transmission torrent client
• Alt+F7 – Toggle on/off transmission client via dmenu
• Alt+F8 – Check mail & RSS
• Alt+F9 – Mount a USB drive/hard drive or Android
• Alt+F10 – Unmount a non-essential drive or Android
• Alt+F11 – View webcam
• Alt+F12 – Rerun keyboard mapping script if new keyboard is attached
Printscreen
• PrtSc – Take a screenshot
• Shift+PrtSc – Select area to screenshot
• Alt+PrtSc – Opens dmenu menu to select kind of audio/video recording
• Alt+Del – Kills any recording started in the above way
• Alt+Shift+PrtSc – Kills any recording started in the above way
• Alt+ScrollLock – Toggle screenkey (if installed) to show keypresses
ST keys (Super/Caps)
• Super+l – Open link
• Super+o – Copy terminal output
• Super+y – Copy link
• Super+c – Cut
• Super+v – Paste
• Super+j – Scroll up
• Super+k – Scroll down
• Super+a – Less transparency
• Super+s – More transparency
• Super+J – Enlage text
• Super+K – Decrease text
ZSH keys
• Ctrl+a – Terminal calculator
• Ctrl+e – Edit command in VIM
• Ctrl+f – Find directory (fzf)
• Ctrl+g – Gallery (C-x+w to set wallpaper)
• Ctrl+o – File explorer

• C-l – Right
