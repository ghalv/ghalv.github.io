Guide: How to turn vim into an R IDE 
,l - run (section)
,e - echo-run section
_ - "left arrow"

\rf - start
\rq - leave
\rv - view data frame
\ro - object browser
\rl - object echo
\rr - clear console
\o - output in file

\vh - head()
\rs - summary()
\rg - plot()
\ra - args()
\rd - setwd()
\rp - print()
\rn - names()

\rh - help
\re - example

Tastatursnarveier til vimrc

~/.config/nvim/init.vim


" remapping the basic :: send line
	nmap  <leader>l  <Plug>RDSendLine
" remapping selection :: send multiple lines
	vmap  <leader>l  <Plug>RDSendSelection
" remapping selection :: send multiple lines + echo lines
	vmap  <leader>e  <Plug>RESendSelection


" KUN hvis dash - remap search key
	nmap <silent> <leader>d <Plug>DashSearch<CR>
